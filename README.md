# 業務管理システム

業務データの正本をスプレッドシート1つに一元化し、日常操作をこのWebアプリで行う。

## 構成

```
[ブラウザ] index.html (GitHub Pages・単一HTML SPA)
     │  起動時に全シートを一括ロード → 検索/絞込み/集計はブラウザ内で即時
     │  書き込みのみ Apps Script Web App へPOST（LockService直列化・ロール検証）
     ▼
[Apps Script Web App] → [業務スプレッドシート]
```

- 利用者はロール切替式（管理者/スタッフ。スタッフは閲覧＋ステータス前進のみ）
- 全画面 CSV出力（UTF-8 BOM）付き。ライト/ダーク両対応・スマホ対応（下タブ）
